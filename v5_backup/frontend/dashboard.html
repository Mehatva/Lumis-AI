Meta Connection Guide

Follow these 3 steps to unlock AI automation for your Instagram.

1

Switch to Professional

Open Instagram App > Settings > Account Type > Professional Account. (Business or Creator).

2

Link Facebook Page

Settings > Professional Tools > Connect/Create Facebook Page. This acts as the API key.

3

Allow Message Access

Settings > Messages and Story Replies > Message Controls > Allow Access to Messages (ON).
